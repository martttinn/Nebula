# Resumen

Grupo que recoge la retirada de los bordes debug visibles en la sección `How we work`.

## Objetivo del grupo

Quitar la superposición de inspección del timeline y devolver el bloque a una presentación limpia.

## Cambios incluidos

- desactivación del flag `DEBUG_BORDERS` en `components/home/how-we-work.tsx`

## Impacto en arquitectura / producto

No cambia la lógica ni la composición final del timeline; elimina solo una capa de depuración visible.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- si vuelve a abrirse una iteración geométrica, habrá que reactivar manualmente el flag o instrumentar otro mecanismo de inspección
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Pulido operativo pequeño sobre una sección pública ya estabilizada.
