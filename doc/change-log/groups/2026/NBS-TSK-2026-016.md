# Resumen

Grupo que recoge el ajuste del navbar para usar el logo oficial `horizontal2` y eliminar las sub-superficies internas.

## Objetivo del grupo

Dejar el navbar viviendo sobre una única cápsula de vidrio, con las tres zonas internas resueltas por layout y no por pills anidados.

## Cambios incluidos

- sustitución del logomark custom por el logo oficial `horizontal2` de `public/logo/horizontal2/`
- retirada de outlines y fondos internos en las tres zonas del navbar
- normalización del borde del shell para mantener un grosor visual único y consistente
- alineación del canon visual y de la documentación técnica con el nuevo estado

## Impacto en arquitectura / producto

La primitive sigue siendo reusable y tripartita, pero su contrato visual se simplifica hacia una sola superficie global más coherente con la dirección glass del proyecto.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- la lectura del branding depende ahora más del contraste real del fondo que atraviesa el vidrio

## Notas para presupuesto

Refinamiento de primitive global existente con alineación documental y sustitución por asset oficial ya disponible en el repo.
