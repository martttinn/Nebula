# Resumen

Grupo que recoge la corrección visual del pulso de `StarBorder` para igualar mejor la lectura del componente de referencia cuando se usa con interior transparente.

## Objetivo del grupo

Evitar que el destello lila invada el contenido interior del CTA `Contactar` manteniendo el acabado outlined sin fondo del navbar.

## Cambios incluidos

- tipado de la custom property usada para el grosor del borde
- recorte de la banda visible del pulso animado en `StarBorder`
- actualización del canon visual y de la referencia técnica asociada

## Impacto en arquitectura / producto

La primitive reusable conserva su API actual y mejora su fidelity visual respecto a React Bits en un caso de uso clave de marca.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- si se reutiliza `StarBorder` con grosores o alturas muy distintos, la banda visible del pulso podría necesitar otro ajuste fino

## Notas para presupuesto

Corrección visual pequeña sobre una primitive reusable y su documentación asociada.
