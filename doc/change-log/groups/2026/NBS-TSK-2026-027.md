# Resumen

Grupo que recoge el ajuste del icono del sitio para alinear la metadata pública con el símbolo oficial oscuro de Nebula.

## Objetivo del grupo

Sustituir el favicon heredado por el símbolo `dark` de marca usando la convención actual del App Router.

## Cambios incluidos

- eliminación de `app/favicon.ico`
- alta de `app/icon.svg`
- actualización de la referencia técnica
- trazabilidad en `change-log`

## Impacto en arquitectura / producto

La percepción de marca queda alineada desde la pestaña del navegador y se simplifica la gestión del icono con metadata file-based.

## Validación agregada

- `npm run build`

## Riesgos o arrastres

- algunos contextos legacy pueden seguir cacheando el icono anterior hasta vaciar caché del navegador

## Notas para presupuesto

Cambio pequeño de branding técnico con coste muy bajo.
