# Resumen

Grupo que recoge la sincronización del borde del CTA `Contactar` con su nuevo hover lila.

## Objetivo del grupo

Conseguir que relleno interior y línea de borde cambien de tono de forma coherente al interactuar.

## Cambios incluidos

- borde base movido a variable CSS
- borde hover en lila de marca
- transición unificada con el fill interior

## Impacto en arquitectura / producto

El CTA del navbar responde como una sola primitive coherente, y `StarBorder` gana un punto de extensión más limpio para futuros overrides.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; el cambio mantiene el comportamiento por defecto en consumers que no definan variables

## Notas para presupuesto

Refinamiento menor de interacción.
