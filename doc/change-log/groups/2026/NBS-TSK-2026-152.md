# Resumen

Grupo que recoge la limpieza visual de las cards de `services-carousel`, retirando los dots ornamentales internos.

## Objetivo del grupo

Simplificar el chrome de las cards de servicios quitando puntos de color internos que ya no aportaban jerarquía ni claridad.

## Cambios incluidos

- eliminación de los radial gradients ornamentales del fondo de card
- retirada del glow circular de acento en la esquina inferior izquierda
- mantenimiento del `BorderGlow`, la rejilla base y el halo del icono central
- registro de la entrega en `doc/change-log/`

## Impacto en arquitectura / producto

La sección conserva su estructura, su motion y su identidad dark-tech, pero las cards quedan más limpias y menos recargadas. El cambio afecta solo al acabado visual interno de cada card, no al comportamiento del carrusel ni al contenido visible.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run changes:sync`

## Riesgos o arrastres

- si después de quitar estos dots la card se percibe demasiado plana, el siguiente ajuste razonable sería recalibrar el `BorderGlow` o el glow del icono, no reintroducir puntos ornamentales iguales
- no se hizo revisión visual manual en navegador dentro de este hilo

## Notas para presupuesto

Refinamiento frontend de detalle sobre una sección pública existente, con foco en limpieza compositiva y reducción de ruido visual.
