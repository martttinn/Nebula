# Resumen

Grupo que recoge la integración de una primitive reusable `StarBorder` y su aplicación al CTA `Contactar` del navbar.

## Objetivo del grupo

Dar más jerarquía visual al CTA principal de navegación sin convertir el sistema completo de botones en una colección de efectos animados.

## Cambios incluidos

- creación de `components/ui/star-border.tsx`
- creación de `components/ui/star-border.module.css`
- aplicación del efecto al CTA `Contactar` del navbar
- adaptación del color del borde animado a la paleta `lilac / haze / navy`
- alineación del canon visual, la referencia técnica y el `change-log`

## Impacto en arquitectura / producto

El sistema gana una primitive reusable de CTA animado para shells oscuros y la navegación principal refuerza la llamada a la acción comercial.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- si se abusa de `StarBorder` en más superficies, la identidad puede degradarse hacia una estética demasiado efectista

## Notas para presupuesto

Integración controlada de primitive reusable con motion y ajuste visual específico del CTA principal del navbar.
