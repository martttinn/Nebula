# Resumen

Grupo que recoge el rediseño del navbar para acercarlo a la referencia visual aportada por Martín.

## Objetivo del grupo

Transformar el navbar desde una composición de subcápsulas hacia una sola superficie alargada, con links suaves y CTA blanco dominante a la derecha.

## Cambios incluidos

- rediseño de `components/layout/navbar.tsx`
- introducción de la utility `glass-navbar`
- ajuste del canon visual y de la decisión duradera del navbar

## Impacto en arquitectura / producto

La navegación global gana una presencia más limpia y editorial, más cercana al lenguaje de la referencia y mejor integrada como shell superior de alto nivel.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- el navbar sigue usando branding temporal de Nebula, no un logo final
- si se quieren más CTAs o navegación móvil avanzada, hará falta una segunda iteración

## Notas para presupuesto

Trabajo de redirección visual y refinamiento de primitive reusable, con foco en composición, proporción y jerarquía.
