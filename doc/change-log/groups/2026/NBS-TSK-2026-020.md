# Resumen

Grupo que recoge la actualización de la navegación visible del navbar para reflejar la nueva arquitectura de información pública pedida.

## Objetivo del grupo

Sustituir los links anteriores del navbar por la nueva lista de secciones de primer nivel: Servicios, Proyectos, Sobre Nosotros, Precios y Blog.

## Cambios incluidos

- retirada de `Inicio` y `Proceso` del set visible de navegación
- incorporación de `Proyectos`, `Sobre Nosotros`, `Precios` y `Blog`
- reasignación temporal de destinos a anchors `/#...` coherentes con la fase actual del frontend

## Impacto en arquitectura / producto

La navegación visible del shell global se alinea con la nueva IA pública, aunque las secciones de destino todavía no existen en runtime.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- mientras no existan las secciones reales, los hashes solo preparan la estructura de navegación y no entregan contenido aún

## Notas para presupuesto

Ajuste de arquitectura de información y navegación visible en la primitive global del navbar.
