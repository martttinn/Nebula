# Resumen

Grupo que recoge la integración de la barra superior global de progreso de scroll como primitive transversal de la shell pública.

## Objetivo del grupo

Añadir una referencia global de avance del documento sin aumentar el ruido visual ni acoplar esa utilidad a una sección concreta.

## Cambios incluidos

- creación del indicador superior de progreso en `components/layout/scroll-progress-bar.tsx`
- integración de la primitive en `app/layout.tsx` para que viva en toda la shell pública
- ajuste visual a un patrón de scrollbar fino, sin glow y con punta derecha redondeada
- actualización del canon visual y de la referencia técnica para fijar la nueva pieza de layout

## Impacto en arquitectura / producto

El repo gana una primitive global de layout reutilizable y desacoplada del contenido. Producto suma una affordance de orientación transversal coherente con el lenguaje dark-tech.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- la utilidad solo aporta valor cuando el documento tiene recorrido vertical real; en páginas muy cortas no aparece
- cualquier cambio futuro en el shell fijo deberá revisar el stacking de esta barra con navbar, preloader y overlays globales

## Notas para presupuesto

Entrega transversal pequeña pero de alto alcance visual: toca la shell global, añade motion sutil y deja una nueva primitive de UX reutilizable.
