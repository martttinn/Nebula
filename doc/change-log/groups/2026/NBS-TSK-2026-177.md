# Resumen

Grupo que recoge la retirada del subrayado y del brillo blanco en el hover de la información de contacto del footer.

## Objetivo del grupo

Simplificar la microinteracción de email y teléfono para que el hover mantenga feedback visual sin añadir líneas bajo el texto ni destellos blancos en el icono.

## Cambios incluidos

- `components/layout/footer.tsx` elimina el pseudo-subrayado progresivo de los textos de contacto.
- `components/layout/footer.tsx` elimina la capa radial blanca interna del icono en hover/focus.
- `DESIGN.md` actualiza el canon de interacción de contacto del footer.

## Impacto en arquitectura / producto

El cambio es puramente visual. No altera enlaces, datos de contacto, layout, rutas, navegación ni backend.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- smoke visual hover desktop en `http://localhost:3001`
- `npm run changes:sync`

## Riesgos o arrastres

El hover conserva color, borde activo y transformación ligera. Si se quiere eliminar también la sombra lila del icono, debe hacerse como decisión visual separada.

## Notas para presupuesto

Ajuste menor de pulido visual del footer, acotado a una microinteracción.
