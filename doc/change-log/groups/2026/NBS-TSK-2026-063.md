# Resumen

Grupo que recoge la incorporación de una banda de propuesta de valor justo después del hero, con frases grandes y cambio de color palabra a palabra según el scroll.

## Objetivo del grupo

Añadir una segunda capa de autoridad a la home pública de Nebula Studios mediante tres frases diferenciales y una primitive reusable de motion tipográfico guiado por scroll.

## Cambios incluidos

- creación de `components/ui/word-by-word-color-change.tsx`
- creación de `components/home/value-proposition-section.tsx`
- integración de la nueva sección en `app/page.tsx`
- actualización de `DESIGN.md`, referencia técnica y `decisions-log`

## Impacto en arquitectura / producto

La home deja de quedar reducida a `Hero` y gana una segunda sección pública real. Además, la librería UI incorpora una primitive scroll-driven reutilizable para copy corto de alto impacto.

## Validación agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- no se ha ejecutado `npm run changes:sync` porque ese tooling sigue documentado pero no expuesto en `package.json`
- el efecto depende del scroll real de viewport, así que la percepción exacta debe revisarse visualmente también en navegador

## Notas para presupuesto

Entrega visible de frontend con componente reusable, motion guiado por scroll y ajuste del storytelling de la home pública.
