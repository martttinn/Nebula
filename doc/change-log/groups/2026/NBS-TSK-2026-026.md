# Resumen

Grupo que recoge el reajuste fino de la capa de partículas del hero para que se lean mejor sin perder sobriedad.

## Objetivo del grupo

Hacer las partículas más visibles y reconocibles dentro del hero, manteniendo una densidad contenida y un comportamiento atmosférico suave.

## Cambios incluidos

- reducción del blur por partícula
- aumento ligero del número total de partículas
- refuerzo del núcleo y halo en la primitive CSS
- actualización de `DESIGN.md` y de la referencia técnica

## Impacto en arquitectura / producto

El hero mejora su legibilidad visual sin cambiar su arquitectura ni introducir nuevos costes de runtime más allá de unos pocos elementos decorativos adicionales.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- si se siguiera aumentando densidad o brillo, el hero podría empezar a competir con el copy y con `GridScan`

## Notas para presupuesto

Micro-refinamiento visual del shell principal con impacto directo en percepción y coste técnico bajo.
