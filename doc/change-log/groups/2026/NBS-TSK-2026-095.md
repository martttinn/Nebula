# Resumen

Grupo que recoge la corrección de las líneas horizontales fantasma visibles entre secciones en móvil.

## Objetivo del grupo

Eliminar las costuras visibles entre `hero`, `benefits` y `services` sin introducir bordes artificiales.

## Cambios incluidos

- cierre totalmente opaco del overlay inferior del hero
- solape de `1px` en las secciones posteriores para absorber seams de redondeo/composición
- actualización de `DESIGN.md` para fijar este criterio como guardrail visual

## Impacto en arquitectura / producto

La home gana continuidad visual entre bloques y evita artefactos horizontales en móvil.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- si otras secciones nuevas introducen fondos o overlays propios, habrá que vigilar la misma clase de seam en breakpoints móviles

## Notas para presupuesto

Refinamiento visual y corrección de artefacto de composición en la home pública.
