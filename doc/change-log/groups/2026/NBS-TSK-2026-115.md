# Resumen

Grupo que recoge la retirada de la carcasa visual del nodo para los pasos del timeline que usan una imagen en lugar de un icono.

## Objetivo del grupo

Permitir que el asset del nodo se vea sin fondo, borde ni clipping circular, manteniendo intacta la geometría del timeline.

## Cambios incluidos

- detección de nodos basados en imagen
- eliminación condicional del shell circular en desktop
- eliminación condicional del shell circular en mobile
- cambio a `object-contain` para evitar el recorte del asset

## Impacto en arquitectura / producto

No modifica el layout base del proceso ni el comportamiento del SVG. Refina la lectura visual de los nodos ilustrados.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- si la dirección de arte evoluciona hacia nodos ilustrados más grandes, habrá que recalibrar tamaño y separación con las cards
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Pulido visual localizado que mejora la integración de assets propios dentro del sistema de nodos del timeline.
