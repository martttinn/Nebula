# Resumen

Grupo que recoge el ajuste del timeline mobile para alinear la guía vertical con los planetas y mostrar más copy en las cards.

## Objetivo del grupo

Corregir dos fricciones visuales específicas de móvil: el eje descentrado respecto al planeta y el truncado excesivo del texto.

## Cambios incluidos

- nuevo posicionamiento horizontal de la guía vertical mobile
- clamp más generoso para la descripción de cards mobile
- actualización del guardrail visual en `DESIGN.md`

## Impacto en arquitectura / producto

No cambia la estructura del componente ni la lógica del recorrido desktop. Ajusta coherencia y legibilidad en la variante móvil.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- si el tamaño del planeta móvil vuelve a crecer mucho, podría hacer falta reequilibrar separación lateral respecto a la card
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Pulido visual móvil de baja complejidad con impacto directo sobre percepción de calidad y claridad de lectura.
