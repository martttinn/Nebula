# Resumen

Grupo que recoge la eliminación del flash final en la salida del hover de los links del navbar.

## Objetivo del grupo

Evitar la reactivación visual del hover al terminar la transición de salida.

## Cambios incluidos

- retirada del desplazamiento vertical
- conservación de la microescala
- estabilización del transform con `will-change`

## Impacto en arquitectura / producto

La navegación central mantiene su tactilidad sin artefactos en la salida del hover.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; el cambio solo suaviza la microinteracción actual

## Notas para presupuesto

Bugfix menor de interacción.
