# Resumen

Grupo que recoge la correccion del movimiento visible del fondo en la seccion de servicios durante el scroll desktop.

## Objetivo del grupo

Evitar que el fondo de la seccion siga desplazandose mientras el carrusel permanece sticky, para que la seccion se perciba como un bloque realmente fijado.

## Cambios incluidos

- traslado del fondo desktop al interior del mismo shell sticky que contiene heading, cards e indicadores
- limitacion del fondo exterior de la seccion al runtime movil

## Impacto en arquitectura / producto

La experiencia de scroll en desktop se vuelve mas coherente: contenido y fondo comparten el mismo anclaje visual durante el tramo sticky.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- no se ha hecho revision visual automatizada en navegador en esta sesion

## Notas para presupuesto

Correccion puntual de percepcion visual y comportamiento de scroll sobre una surface publica ya integrada.
