# Resumen

Grupo que recoge la incorporación de botones sociales en el footer de Nebula Studios con el patrón visual del footer de Canal3.

## Objetivo del grupo

Añadir accesos sociales visibles y compactos en la columna de marca/contacto sin alterar la estructura de columnas del footer.

## Cambios incluidos

- `lib/site.ts` centraliza las URLs sociales visibles.
- `components/layout/footer.tsx` añade botones de Instagram, LinkedIn y GitHub.
- Los botones usan superficie glass, borde fino, icono centrado, hover por red y apertura externa en nueva pestaña.
- `DESIGN.md` y las referencias documentales recogen el nuevo patrón.

## Impacto en arquitectura / producto

El cambio añade salidas externas de marca al cierre de la home. No cambia rutas internas, CTAs comerciales, formulario, backend ni navegación legal.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- smoke visual desktop en `http://localhost:3001`
- smoke visual móvil en `http://localhost:3001`
- `npm run changes:sync`

## Riesgos o arrastres

Los enlaces sociales son salidas externas y dependen de la disponibilidad de cada plataforma. LinkedIn puede responder con bloqueo anti-bot en comprobaciones HTTP aunque el destino se mantenga como URL pública.

## Notas para presupuesto

Entrega pequeña de UI pública orientada a confianza y presencia de marca, sin integración de captación ni tracking.
