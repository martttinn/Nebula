# Resumen

Grupo que recoge el refinamiento cromático del CTA `Contactar` para alinear `StarBorder` con la jerarquía blanca del CTA y el lila principal de Nebula.

## Objetivo del grupo

Sustituir la versión oscura inicial del CTA animado por una variante de fondo blanco y destello lila de marca, más coherente con la prioridad visual pedida.

## Cambios incluidos

- cambio del relleno del CTA `StarBorder` a blanco
- cambio del color del destello al lila principal de marca
- ajuste del texto del CTA a color oscuro
- alineación del canon visual, decisiones y referencia técnica

## Impacto en arquitectura / producto

La primitive `StarBorder` mantiene su API y su comportamiento, pero cambia su baseline visual hacia una jerarquía más comercial y directa.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- sobre fondos muy claros esta variante no funcionaría igual de bien; hoy está pensada para shells oscuros como el navbar actual

## Notas para presupuesto

Refinamiento cromático y jerárquico de una primitive reusable ya integrada en el runtime público.
