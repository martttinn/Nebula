# Resumen

Se crea una primitive reutilizable para mostrar logotipos externos desde una carpeta pública canónica.

## Objetivo del grupo

Centralizar el rendering de logos de marcas, tecnologías o clientes para evitar paths, tamaños y reglas de accesibilidad repetidas en futuras secciones.

## Cambios incluidos

- `NBS-CHG-2026-05-19-196`: componente `BrandLogo` y carpeta pública `brand-logos`.

## Impacto en arquitectura / producto

El cambio afecta a la capa de componentes compartidos y al sistema visual documentado. No altera la home actual ni monta nuevos logos en runtime.

## Validacion agregada

- `npm run typecheck`
- `npm run lint`

## Riesgos o arrastres

`npm run design:lint` está mencionado en reglas internas, pero no existe como script real en `package.json`; queda reportado como limitación de tooling.

## Notas para presupuesto

Entrega de bajo alcance visual directo pero con valor estructural: reduce duplicación futura y prepara una primitive reusable para logos de tecnologías, clientes y marcas externas.
