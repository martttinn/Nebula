# Resumen

Grupo que recoge la sustitución del icono del primer nodo del timeline por una imagen de planeta.

## Objetivo del grupo

Acercar el primer paso del proceso al lenguaje visual espacial de la marca sin alterar la primitive general del timeline.

## Cambios incluidos

- ampliación del modelo de datos del paso para soportar imagen
- sustitución del icono del nodo `01` por `/planets/planet-1.png`
- soporte de render condicional con `next/image` en desktop y mobile

## Impacto en arquitectura / producto

No modifica estructura ni comportamiento del timeline. Refuerza el lenguaje visual y deja preparada la primitive para futuras sustituciones de iconos por imágenes.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- mezclar imágenes y glyphs puede requerir calibración visual adicional si se sustituyen más nodos
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Refinamiento visual pequeño con ligera ampliación de la primitive del nodo para soportar assets de marca.
