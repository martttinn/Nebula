# Resumen

Grupo que recoge la eliminación del clipping vertical restante en el hover de los links del navbar.

## Objetivo del grupo

Hacer que el highlight pill del hover pueda renderizarse completo en vertical.

## Cambios incluidos

- retirada del `overflow-hidden` intermedio
- eliminación del margen vertical negativo en el rail
- conservación del padding vertical efectivo

## Impacto en arquitectura / producto

La navegación central deja de cortar el highlight sin volver a artefactos previos de flash ni a una interacción más rígida.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; el cambio está confinado al wrapper de links del navbar

## Notas para presupuesto

Bugfix menor de presentación.
