# Resumen

Grupo que recoge la primera pieza audiovisual editorial construida sobre el subsistema `remotion/` de Nebula Studios.

## Objetivo del grupo

Pasar de una integración técnica mínima de Remotion a una composición real de branding que sirva como intro cinematográfica del negocio y de la web pública.

## Cambios incluidos

- creación de la composición `NebulaEditorialIntro`
- extracción de una utilidad compartida para la carga segura de tipografías en Remotion
- registro de la nueva composición en `remotion/Root.tsx`
- validación mediante stills de apertura, tramo medio y cierre, además de render MP4 final
- actualización de la referencia técnica y del `change-log`

## Impacto en arquitectura / producto

No hay impacto en producción ni en rutas públicas. El subsistema de vídeo gana su primera pieza editorial usable como activo de branding interno o material de presentación.

## Validacion agregada

- `npx remotion compositions remotion/index.ts`
- `npm run video:still -- NebulaEditorialIntro renders/nebula-editorial-intro-open.png --frame 42`
- `npm run video:still -- NebulaEditorialIntro renders/nebula-editorial-intro-middle.png --frame 176`
- `npm run video:still -- NebulaEditorialIntro renders/nebula-editorial-intro-close.png --frame 314`
- `npm run video:render -- NebulaEditorialIntro renders/nebula-editorial-intro.mp4`
- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run changes:sync`

## Riesgos o arrastres

- la composición nace sin audio y sin voiceover; eso es deliberado para mantener la v1 centrada en identidad visual y ritmo editorial
- si más adelante se reutiliza esta pieza para campañas o landings concretas, convendrá parametrizar copy, timing y assets antes de seguir duplicando composiciones

## Notas para presupuesto

Entrega creativa con valor visible: no añade solo tooling, sino un activo audiovisual real alineado con el posicionamiento premium de Nebula y preparado para iterarse hacia piezas comerciales más complejas.
