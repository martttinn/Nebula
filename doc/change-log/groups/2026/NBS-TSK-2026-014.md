# Resumen

Grupo que recoge el refuerzo del comportamiento de vidrio real en la primitive global del navbar.

## Objetivo del grupo

Hacer que el navbar no solo sea translúcido, sino que se perciba inequívocamente como una superficie `frosted glass` con blur, highlights y profundidad.

## Cambios incluidos

- aumento del blur y de la respuesta óptica del shell
- reordenación de capas para que los contenidos del navbar queden por encima de los efectos
- actualización del canon visual del componente

## Impacto en arquitectura / producto

La primitive global del navbar mejora su legibilidad como material visual y se alinea mejor con la referencia compartida por Martín.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- sigue dependiendo del fondo posterior para ofrecer su mejor lectura

## Notas para presupuesto

Refinamiento de alta sensibilidad visual sobre una primitive reusable del sistema de navegación.
