# Resumen

Grupo que recoge la reintegración de `testimonials` en la home de Nebula Studios tomando de Golden Grama el patrón de stack sticky y manteniendo la estética propia de Nebula.

## Objetivo del grupo

Recuperar la prueba social visible en la home, pero abandonando la primitive anterior de Nebula y reconstruyéndola con el flujo sticky de Golden Grama sin perder la identidad visual de Nebula.

## Cambios incluidos

- creación de una nueva implementación de `components/home/testimonials/`
- uso de `archive.ts` como fuente repo-safe de las tres reseñas verificadas
- integración otra vez de `TestimonialsSection` en `app/page.tsx`
- replicación del patrón sticky stack de cards que Golden Grama usa en `ServicesClient`
- actualización de `DESIGN.md`, referencias técnicas, dominio y `decisions-log`
- registro de la entrega en `doc/change-log/`

## Impacto en arquitectura / producto

La home vuelve a incluir una surface de confianza después de `Projects showcase`, ahora con una primitive nueva que conserva el sistema visual de Nebula y solo adopta de Golden Grama la mecánica de cards apiladas por scroll. La estructura mantiene el contenido tipado y desacoplado de la UI para futuras iteraciones.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run changes:sync`

## Riesgos o arrastres

- Golden Grama no expone una `TestimonialsSection` standalone; la referencia externa útil ha sido su patrón sticky de cards en `ServicesClient`, no una surface de prueba social completa lista para copiar

## Notas para presupuesto

Trabajo visible y de diseño aplicado: reconstrucción de una sección pública a partir de una referencia externa interna del propio ecosistema de Martín, con extracción de layout, motion, tipografía y chrome editorial.
