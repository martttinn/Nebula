# Resumen

Grupo que recoge el ajuste de tangencia del path SVG al pasar por los nodos internos del timeline desktop de `How we work`.

## Objetivo del grupo

Hacer que la linea atraviese los hitos intermedios por un eje vertical limpio, en vez de rozarlos con una tangencia oblicua.

## Cambios incluidos

- anclaje del `x` de los handles de entrada y salida al eje del nodo en puntos internos del path

## Impacto en arquitectura / producto

No cambia la estructura de la home ni el layout de la seccion. Refina la geometria del SVG y la lectura visual del timeline.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- la compilacion y el tipado estan verificados, pero la aprobacion final depende de revision visual sobre la composicion real
- `DEBUG_BORDERS` sigue activo y puede contaminar parcialmente la lectura final mientras no se desactive
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Pulido visual pequeño pero visible sobre una seccion publica de alto peso narrativo.
