# Resumen

Grupo que recoge la corrección del `StarBorder` del CTA `Contactar` para que el efecto vuelva a leerse con claridad tras los últimos ajustes visuales.

## Objetivo del grupo

Recuperar la visibilidad real del borde animado lila sin abandonar la superficie blanca pedida para el CTA principal del navbar.

## Cambios incluidos

- aumento del grosor efectivo del contenedor `StarBorder`
- ampliación del área visible de los gradientes animados
- relajación del borde estático interno para que no compita con el destello
- ajuste del CTA `Contactar` a `thickness={2}`

## Impacto en arquitectura / producto

La primitive conserva su API, pero mejora su baseline visual y recupera su función de acento premium en la navegación principal.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- si en el futuro se reduce otra vez el grosor o se cambia a superficies muy luminosas, el efecto puede volver a perder lectura

## Notas para presupuesto

Corrección visual puntual de primitive reusable ya integrada en el runtime público.
