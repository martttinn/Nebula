# Resumen

Grupo que recoge la correccion cromatica de textos descriptivos en servicios, proyectos y testimonials.

## Objetivo del grupo

Restaurar el blanco suave en descripciones y subtitulos que se habian desplazado visualmente hacia lila tras la actualizacion del stack.

## Cambios incluidos

- Ajuste del subtitulo de cards de servicios.
- Ajuste de la descripcion de proyectos en desktop y mobile.
- Ajuste del resumen de testimonials en desktop y mobile.

## Impacto en arquitectura / producto

Impacto visual local en componentes publicos. No altera copy, datos, rutas ni comportamiento.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- Playwright computed styles servicios/proyectos/testimonials
- `npm run changes:sync`

## Riesgos o arrastres

Quedan intactos labels y metadatos que siguen usando `Haze` o `Lilac` de forma intencional.

## Notas para presupuesto

Correccion fina de drift visual posterior a migracion de Tailwind/stack base.
