# Resumen

Grupo que recoge el ajuste del CTA principal del navbar para eliminar su relleno y dejar un `StarBorder` outlined real.

## Objetivo del grupo

Preservar el borde animado lila del CTA `Contactar` sin convertir el botón en una superficie rellena dentro del shell glass del navbar.

## Cambios incluidos

- transparencia total de la variante `outline` en `StarBorder`
- ajuste del uso concreto dentro del navbar
- actualización de `DESIGN.md`, decisiones y referencia técnica

## Impacto en arquitectura / producto

El navbar gana un CTA más ligero y coherente con la idea de cápsula outlined premium, y la primitive reusable queda mejor definida semánticamente.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`
