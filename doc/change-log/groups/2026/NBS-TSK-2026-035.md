# Resumen

Grupo que recoge el endurecimiento visual de la capa de partículas del hero hacia círculos lilas definidos y sin desenfoque.

## Objetivo del grupo

Eliminar cualquier lectura de mota difusa y hacer que cada partícula sea reconocible como círculo.

## Cambios incluidos

- eliminación del blur en datos y estilos
- eliminación de halos y gradientes suaves
- ajuste documental del canon visual y de la referencia técnica

## Impacto en arquitectura / producto

La capa atmosférica del hero mantiene su deriva lenta, pero adopta una estética más gráfica y definida.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- al no tener blur ni glow, algunas partículas pequeñas pueden percibirse algo más duras; si hiciera falta, se corregirá por tono u opacidad

## Notas para presupuesto

Refinamiento visual menor sobre componente decorativo.
