# Resumen

Grupo que recoge la conversión de la variante móvil de la sección de servicios desde un carrusel horizontal a una lista vertical de cards completas.

## Objetivo del grupo

Eliminar fricción de navegación en móvil y permitir scroll vertical nativo sobre toda la sección sin perder la identidad visual de las cards.

## Cambios incluidos

- sustitución del carrusel horizontal móvil por un stack vertical de cards
- extracción de un shell visual reutilizable para compartir chrome entre desktop y móvil
- mantenimiento del arco sticky actual en desktop
- actualización de `DESIGN.md`, referencia técnica y decisiones operativas

## Impacto en arquitectura / producto

La sección conserva su showcase diferencial en desktop, pero en móvil prioriza legibilidad y navegación natural. Desaparece la dependencia de swipe horizontal para descubrir servicios.

## Validacion agregada

- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- conviene revisar visualmente el ritmo vertical entre cards en breakpoints intermedios para confirmar que el stack no se siente demasiado denso

## Notas para presupuesto

Refactor responsive de una sección pública para reducir fricción de uso en móvil sin reabrir la variante desktop.
