# Resumen

Grupo que recoge la retirada de la primitive `DecryptedText` y de sus artefactos asociados al no tener ya uso real en la web actual.

## Objetivo del grupo

Reducir superficie muerta en la librería UI local y alinear la referencia técnica con el runtime real.

## Cambios incluidos

- eliminación de `components/ui/decrypted-text.tsx`
- eliminación de `components/ui/decrypted-text.module.css`
- limpieza de la referencia técnica viva del repo

## Impacto en arquitectura / producto

La capa UI queda más estrecha y consistente con las primitives realmente activas. El histórico se conserva en `change-log`, pero la documentación operativa deja de presentar `DecryptedText` como opción vigente.

## Validación agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- no se ha ejecutado `changes:sync` porque esos scripts no están expuestos hoy en `package.json`

## Notas para presupuesto

Limpieza técnica y documental orientada a reducir deuda de mantenimiento y evitar confusión futura en la librería de primitives.
