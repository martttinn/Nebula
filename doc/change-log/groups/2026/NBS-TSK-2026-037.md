# Resumen

Grupo que recoge la unificación del lenguaje de aparición del copy principal del hero.

## Objetivo del grupo

Hacer que heading y subheading compartan el mismo tipo de reveal, manteniendo la secuencia actual y la jerarquía visual.

## Cambios incluidos

- sustitución del subheading a `SplitText`
- retirada del fade auxiliar del párrafo
- actualización documental del runtime y del canon visual

## Impacto en arquitectura / producto

La home gana una entrada tipográfica más coherente al reducir la mezcla de primitives de motion en el bloque principal.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- si en móvil el reveal del subheading se sintiera demasiado denso, convendría bajar aún más su delay antes que cambiar otra vez de primitive

## Notas para presupuesto

Refinamiento menor de coherencia visual.
