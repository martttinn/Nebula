# Resumen

Grupo que recoge el ajuste tipografico del texto de cita dentro de la seccion `Testimonials`.

## Objetivo del grupo

Hacer que el quote de las reseñas use la tipografia secundaria del sistema, manteniendo intacta la composicion general de la seccion.

## Cambios incluidos

- sustitucion de `font-display` por `font-sans` en las citas desktop
- sustitucion de `font-display` por `font-sans` en las citas mobile
- registro de la entrega en `doc/change-log/`

## Impacto en arquitectura / producto

La seccion `Testimonials` mantiene su estructura y su narrativa, pero alinea el quote principal con la tipografia secundaria del sistema visual. El cambio es local y visible al usuario.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npm run changes:sync`

## Riesgos o arrastres

- no se realizo revision visual manual en navegador dentro de este hilo
- si despues se busca un ajuste editorial mas fino, puede hacer falta retocar tracking o leading especificamente para `Inter`

## Notas para presupuesto

Ajuste visual localizado en una seccion publica con impacto directo en coherencia tipografica y legibilidad editorial.
