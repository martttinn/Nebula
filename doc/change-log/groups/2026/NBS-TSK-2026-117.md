# Resumen

Grupo que recoge una segunda ampliación del tamaño del nodo ilustrado dentro del timeline de proceso.

## Objetivo del grupo

Llevar el planeta a una escala más protagonista sin arrastrar ese cambio al resto de nodos del sistema.

## Cambios incluidos

- aumento de `IMAGE_NODE_DESKTOP_SIZE`
- aumento de `IMAGE_NODE_MOBILE_SIZE`
- conservación del comportamiento exclusivo para nodos con imagen

## Impacto en arquitectura / producto

No altera layout estructural ni geometría del SVG. Ajusta jerarquía visual del primer cluster.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- una nueva ampliación podría exigir más aire respecto a la card
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Pulido visual incremental orientado a reforzar el lenguaje espacial de marca.
