# Resumen

Grupo que recoge la incorporacion de logos de empresa en las cards de testimonials de la home publica.

## Objetivo del grupo

Mostrar marca de empresa junto al nombre de empresa en testimonials, separando claramente `Empresa` y `Cargo` para conservar legibilidad.

## Cambios incluidos

- Soporte de PNG en `BrandLogo`.
- Metadatos `companyLogo` para Golden Grama y Future Nova.
- Reestructuracion de campos de empresa/cargo en testimonials desktop y mobile.
- Actualizacion del canon visual para documentar logos SVG o PNG.

## Impacto en arquitectura / producto

Impacto acotado a UI publica y assets estaticos. Mejora la prueba social de la home sin introducir nuevas dependencias ni backend.

## Validacion agregada

- `file public/brand-logos/fn.png public/brand-logos/gg.png`
- `sips -g pixelWidth -g pixelHeight public/brand-logos/fn.png public/brand-logos/gg.png`
- `npm run lint`
- `npm run typecheck`
- `npm run build`
- Playwright smoke visual desktop/mobile testimonials
- `npm run changes:sync`

## Riesgos o arrastres

Canal3Networks queda sin logo hasta que exista un asset canonico en `public/brand-logos`.

## Notas para presupuesto

Mejora visual de testimonials con integracion de assets reales de cliente/proyecto y ajuste responsive de layout.
