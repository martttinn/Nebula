# Resumen

Grupo que recoge la sustitución del símbolo central de las cards del carrusel de servicios por el icono 3D servido desde `public/3d-Icons`.

## Objetivo del grupo

Usar el asset 3D disponible en el repo como pieza visual central de las cards de servicios y diferenciar esta sección del uso más identitario del símbolo de Nebula.

## Cambios incluidos

- reemplazo del símbolo oficial de Nebula por `public/3d-Icons/iphone-icon-3d.png` en `components/home/services-carousel.tsx`
- reajuste menor del tamaño del bloque visual central para acomodar el nuevo asset cuadrado
- actualización de `DESIGN.md`, referencia técnica y `decisions-log`

## Impacto en arquitectura / producto

La sección de servicios conserva su estructura y mecánica, pero cambia el objeto visual protagonista de cada card por un icono 3D específico del catálogo de assets públicos.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- todas las cards comparten hoy el mismo icono 3D; si más adelante se quiere mayor diferenciación por servicio, hará falta ampliar el catálogo de assets

## Notas para presupuesto

Refinamiento visual de una sección pública clave mediante reutilización de asset 3D existente en el repositorio.
