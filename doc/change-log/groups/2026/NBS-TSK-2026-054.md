# Resumen

Grupo que recoge un nuevo ajuste del campo de partículas del hero hacia puntos más pequeños y completamente opacos en su pico visible.

## Objetivo del grupo

Reducir aún más el diámetro aparente de las partículas y llevar su opacidad máxima al 100% para reforzar la lectura del campo.

## Cambios incluidos

- reducción adicional del tamaño renderizado
- opacidad máxima fijada al 100% durante la fase visible

## Impacto en arquitectura / producto

La primitive mantiene el mismo patrón de deriva y desvanecimiento, pero el balance visual se mueve hacia puntos más finos y más nítidos.

## Validación agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- con opacidad total, algunas partículas pueden competir más con el copy sobre ciertas zonas del fondo; si ocurre, el siguiente ajuste razonable sería modular la opacidad por tamaño o por región

## Notas para presupuesto

Refinamiento visual menor sobre el hero principal.
