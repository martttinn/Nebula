# Resumen

Grupo que recoge la retirada del gradiente `Navy` en la banda de propuesta de valor para eliminar la costura visible con el hero.

## Objetivo del grupo

Hacer que la transición entre hero y benefits continúe sobre la misma base `Void`, sin abrir una banda cromática separada.

## Cambios incluidos

- retirada del gradiente vertical `Void -> Navy -> Void` en `value-proposition-section`
- base de la sección fijada en `Void` con solo un acento radial lila tenue
- actualización del canon visual y de la referencia técnica

## Impacto en arquitectura / producto

La mecánica scroll-driven de la sección no cambia; se corrige únicamente la continuidad material entre superficies públicas contiguas.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- no se ha hecho revisión visual automatizada en navegador en esta sesión

## Notas para presupuesto

Refinamiento visual localizado de continuidad entre dos superficies públicas de alto impacto.
