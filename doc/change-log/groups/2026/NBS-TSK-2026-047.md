# Resumen

Grupo que recoge el refinamiento de hover con inercia suave para los links centrales del navbar.

## Objetivo del grupo

Dar a la navegación principal una respuesta más premium y menos brusca al pasar el cursor por encima.

## Cambios incluidos

- transición más larga
- curva elástica contenida
- microelevación
- microescala
- ajuste sutil del fondo hover

## Impacto en arquitectura / producto

La navegación central gana coherencia con el lenguaje de movimiento del navbar sin robar protagonismo al CTA.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; el cambio está acotado al navbar actual de escritorio

## Notas para presupuesto

Refinamiento menor de interacción.
