# Resumen

Grupo que recoge el montaje del navbar reusable sobre la página home.

## Objetivo del grupo

Hacer visible en la home el navbar global pill creado previamente, sin subir todavía esa navegación al layout raíz del proyecto.

## Cambios incluidos

- integración de `Navbar` en `app/page.tsx`
- actualización de la referencia técnica para reflejar el runtime actual
- registro de la entrega en `doc/change-log/`

## Impacto en arquitectura / producto

La home pasa a exponer ya la navegación superior reusable, mientras el componente sigue desacoplado del layout global para no imponer todavía comportamiento en futuras rutas.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- el navbar ya es visible en la home, pero sus destinos siguen siendo provisionales mientras no se cierre el mapa final de secciones y rutas

## Notas para presupuesto

Integración pequeña pero visible: reutiliza una primitive de shell ya creada y la conecta al runtime principal del sitio.
