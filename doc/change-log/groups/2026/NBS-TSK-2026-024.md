# Resumen

Grupo que recoge la simplificación del área de CTAs del navbar y la transición del `StarBorder` a una variante outlined.

## Objetivo del grupo

Dejar el navbar con un único CTA `Contactar` y hacer que ese CTA se lea como outlined sin perder el efecto animado lila.

## Cambios incluidos

- retirada del CTA `Explorar` del navbar
- mantenimiento exclusivo del CTA `Contactar`
- incorporación de una variante `outline` en la primitive `StarBorder`
- alineación del canon visual, decisiones y referencia técnica con el nuevo estado

## Impacto en arquitectura / producto

La primitive `StarBorder` gana una variante reutilizable más coherente con shells oscuros y el navbar reduce ruido de acción secundaria.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- si en el futuro se quisiera recuperar un CTA secundario en el navbar, habría que reequilibrar el peso visual entre links y acción principal

## Notas para presupuesto

Refinamiento del shell global de navegación y generalización controlada de una primitive reusable.
