# Resumen

Grupo que recoge la simplificación final del hover del CTA principal del navbar.

## Objetivo del grupo

Dejar el estado interactivo del botón `Contactar` reducido al fill lila en hover, sin escalado.

## Cambios incluidos

- eliminación de la escala hover
- mantenimiento del estado base transparente
- actualización documental del comportamiento final

## Impacto en arquitectura / producto

El CTA mantiene presencia visual pero con una interacción más sobria y predecible.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; si hiciera falta más respuesta visual, deberá venir de color o borde, no de tamaño

## Notas para presupuesto

Refinamiento menor de microinteracción.
