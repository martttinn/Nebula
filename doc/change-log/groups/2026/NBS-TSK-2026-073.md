# Resumen

Grupo que recoge la conversion de las cards del carrusel de servicios a un comportamiento visual totalmente opaco.

## Objetivo del grupo

Evitar que las cards se lean como superficies translúcidas, incluso en estados laterales o de overlap.

## Cambios incluidos

- retirada de la modulacion de opacidad del arco
- fondo base de card convertido a opaco
- actualizacion del contrato visual de `services-carousel` en `DESIGN.md`

## Impacto en arquitectura / producto

La profundidad del carrusel deja de depender de transparencia y pasa a construirse solo con escala, rotacion y orden visual.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- el repo no expone `npm run design:lint`
- no se ha hecho revision visual automatizada en navegador en esta sesion

## Notas para presupuesto

Refinamiento visual localizado del carrusel de servicios con ajuste de primitive documentada.
