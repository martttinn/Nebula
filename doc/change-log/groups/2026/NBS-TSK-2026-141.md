# Resumen

Grupo que recoge la reducción del tamaño visible de las partículas del hero para contener mejor su peso visual.

## Objetivo del grupo

Mantener la atmósfera espacial del hero con una lectura más fina y menos invasiva alrededor del copy principal.

## Cambios incluidos

- reducción del diámetro renderizado de las partículas al `72%` del tamaño base
- conservación íntegra de rutas, opacidad y ciclo de vida definidos en el ajuste orgánico anterior
- corrección del `change-log` para separar este refinamiento del grupo histórico del CTA con el que había quedado mezclado por colisión de ID

## Impacto en arquitectura / producto

Ajuste localizado en `HeroParticles`. Producto gana una capa atmosférica más sutil y el histórico documental recupera una agrupación fiel al cambio ejecutado.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- si en el futuro se busca todavía más control visual, la siguiente evolución razonable sería diferenciar tamaños por familias de partículas o por breakpoint

## Notas para presupuesto

Refinamiento visual menor sobre una primitive decorativa del hero.
