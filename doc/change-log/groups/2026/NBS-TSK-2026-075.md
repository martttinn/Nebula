# Resumen

Grupo que recoge la retirada total del efecto de hover en las cards del carrusel de servicios.

## Objetivo del grupo

Eliminar la respuesta al puntero para que la interacción del carrusel dependa solo del scroll y del snap.

## Cambios incluidos

- retirada del `whileHover` sobre la shell de `ServiceCard`
- limpieza del contrato del componente para eliminar el prop `reducedMotion` ya innecesario
- actualizacion del contrato visual de `services-carousel` en `DESIGN.md`

## Impacto en arquitectura / producto

Las cards dejan de alterar su geometría por hover y pasan a comportarse como piezas puramente guiadas por la navegación del carrusel.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- el repo no expone `npm run design:lint`
- no se ha hecho revision visual automatizada en navegador en esta sesion

## Notas para presupuesto

Refinamiento localizado de interacción para hacer más estable y coherente una sección pública crítica.
