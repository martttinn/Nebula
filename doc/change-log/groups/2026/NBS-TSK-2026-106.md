Grupo que recoge el ajuste de handle vertical para que la linea salga del primer nodo del timeline con una caida mas natural.

Evitar que el tramo `01 -> 02` se hunda demasiado pronto nada mas abandonar el primer nodo real del recorrido.

## Cambios incluidos

- introduccion de una ratio vertical especifica para nodos reales exteriores del timeline

## Impacto en arquitectura / producto

No cambia la estructura de la home ni el layout del timeline. Refina la geometria del SVG y la lectura visual del primer tramo.

## Validacion agregada

- `npm run lint`
- `npm run build`
