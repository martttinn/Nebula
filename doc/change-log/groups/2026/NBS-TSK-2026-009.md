# Resumen

Grupo que recoge la creación de un navbar global reusable para Nebula Studios.

## Objetivo del grupo

Definir una primitive global de navegación con formato pill alargado, tres zonas internas y glassmorphism sobrio, preparada para reutilizarse en futuras rutas.

## Cambios incluidos

- creación de `components/layout/navbar.tsx`
- introducción de las utilidades visuales `glass-pill` y `no-scrollbar`
- actualización de `DESIGN.md`, referencia técnica y decisiones duraderas

## Impacto en arquitectura / producto

El repo gana un shell global reutilizable para navegación de alto nivel, separado de la home y alineado con el lenguaje visual premium del proyecto.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- el componente queda preparado pero no acoplado todavía a un mapa final de rutas productivas
- si el IA o las rutas cambian, habrá que revisar destinos y copy de enlaces

## Notas para presupuesto

Trabajo de sistema visual y frontend reusable: no es solo maquetación, también establece una boundary global para navegación futura.
