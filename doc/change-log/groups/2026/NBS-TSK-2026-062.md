# Resumen

Grupo que recoge la retirada del efecto `StarBorder` del CTA `Contactar` del navbar y la simplificación de la capa UI resultante.

## Objetivo del grupo

Eliminar el borde animado del CTA principal del navbar y dejar ese botón resuelto con la primitive estándar compartida, reduciendo superficie específica sin valor suficiente.

## Cambios incluidos

- sustitución de `StarBorder` por `Button` en `components/layout/navbar.tsx`
- eliminación de `components/ui/star-border.tsx` y `components/ui/star-border.module.css`
- actualización de `DESIGN.md`, referencia técnica y `decisions-log`

## Impacto en arquitectura / producto

El navbar mantiene su jerarquía y su comportamiento principal, pero la librería UI pierde una primitive ya innecesaria y el sistema de botones queda más simple alrededor de una sola base reusable.

## Validación agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- no se ha ejecutado `npm run changes:sync` porque ese tooling sigue documentado pero no expuesto en `package.json`

## Notas para presupuesto

Limpieza estructural con efecto visible en la UI pública y reducción de deuda de mantenimiento en la capa de primitives.
