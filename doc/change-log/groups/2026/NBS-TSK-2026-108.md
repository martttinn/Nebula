# Resumen

Grupo que recoge la simplificación de la jerarquía interna de las cards de `How we work`.

## Objetivo del grupo

Reducir ruido visual dentro de cada card y estabilizar la altura del texto secundario con un máximo visible de dos líneas.

## Cambios incluidos

- retirada del eyebrow en desktop y móvil
- clamp de la descripción a dos líneas en ambas variantes

## Impacto en arquitectura / producto

No cambia la estructura de la home ni la geometría del timeline. Refina el contenido visible dentro de las cards del proceso.

## Validacion agregada

- `npm run lint`
- `npm run build`

## Riesgos o arrastres

- si más adelante alguna descripción necesita más contexto visible, habrá que revisar el clamp o el copy
- `DEBUG_BORDERS` sigue activo y puede contaminar la lectura final mientras no se desactive
- `npm run changes:sync` no puede ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`

## Notas para presupuesto

Pulido visual pequeño pero visible sobre una sección editorial clave de la home.
