# Resumen

Grupo que recoge la retirada del subtitulo visible de las cards de testimonials en la home publica.

## Objetivo del grupo

Simplificar la lectura de cada reseña para que la card muestre solo el claim principal, sin contexto secundario bajo la cita.

## Cambios incluidos

- Retirada del texto `summary` renderizado bajo el claim en desktop y mobile.
- Actualizacion de `DESIGN.md` para reflejar que la reseña visible usa un unico nivel editorial.

## Impacto en arquitectura / producto

Impacto visual local en la seccion publica de testimonials. No cambia datos, rutas, motion base, backend ni dependencias.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`
- Playwright DOM check testimonials desktop/mobile
- `npm run changes:sync`

## Riesgos o arrastres

El campo `summary` permanece en el archivo de reseñas como contenido repo-safe no renderizado, por si se reutiliza en otra variante posterior.

## Notas para presupuesto

Ajuste fino de jerarquia visual en prueba social, con sincronizacion del canon visual.
