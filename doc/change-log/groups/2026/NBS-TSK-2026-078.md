# Resumen

Grupo que recoge la retirada del chevron de `Servicios` y la reversión del navbar desktop al estado inline simple anterior a cualquier dropdown o expansión integrada.

## Objetivo del grupo

Recuperar una cabecera estable, limpia y predecible después de descartar que la preview integrada de servicios alcanzara el nivel visual y de transición esperado.

## Cambios incluidos

- simplificación de `components/layout/navbar.tsx` para volver a navegación inline simple en desktop
- retirada del estado expandido y de toda la mecánica de chevron / panel de servicios
- actualización de `DESIGN.md`, referencia técnica y `decisions-log`

## Impacto en arquitectura / producto

La navegación pública reduce complejidad y vuelve a una sola altura estable en desktop. La sección de servicios mantiene en exclusiva la exposición detallada del catálogo del estudio.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- se pierde la preview rápida de servicios desde la cabecera desktop
- no se ha regenerado el índice global del `change-log` porque el tooling `changes:sync` sigue sin estar expuesto en `package.json`

## Notas para presupuesto

Reversión de experimento de navegación pública y limpieza de complejidad UI no consolidada.
