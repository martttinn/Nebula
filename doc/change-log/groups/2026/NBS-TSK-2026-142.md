# Resumen

Grupo que recoge el incremento del brillo de las partículas del hero tras reducir su tamaño visible.

## Objetivo del grupo

Recuperar presencia visual en la capa de partículas sin volver a crecerlas ni introducir glow o blur adicionales.

## Cambios incluidos

- aclarado del color base de las partículas hacia un lila más luminoso
- incremento moderado de opacidad efectiva con tope controlado
- saneamiento del `change-log` para aislar este ajuste del grupo del CTA con el que había colisionado en el histórico

## Impacto en arquitectura / producto

Ajuste localizado en `HeroParticles`, sin cambiar rutas ni timings. Producto mantiene una atmósfera nítida, pero más legible sobre el hero.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- si se necesita más contraste en el futuro, el siguiente paso razonable sería separar partículas protagonistas y secundarias en dos niveles de luminosidad

## Notas para presupuesto

Refinamiento visual menor sobre el hero principal.
