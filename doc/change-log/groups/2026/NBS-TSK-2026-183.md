# Resumen

Grupo que recoge la actualización mayor del stack base posterior al patch de Next.

## Objetivo del grupo

Actualizar React, Tailwind, TypeScript y dependencias visuales/tooling manteniendo la web pública estable y sin drift visual intencional.

## Cambios incluidos

- React 19 y tipos React 19.
- Tailwind CSS 4 con `@tailwindcss/postcss`.
- Configuración Tailwind migrada de TypeScript a CommonJS para evitar warnings ESM.
- Adaptaciones de compatibilidad visual para utilidades renombradas de Tailwind 4.
- Three, Remotion, Supabase CLI y TypeScript actualizados.
- ESLint 10 probado y descartado por incompatibilidad real con `eslint-config-next`.
- Documentación técnica y change-log actualizados.

## Impacto en arquitectura / producto

Impacto transversal en runtime frontend, pipeline CSS y tooling. La superficie pública mantiene el mismo objetivo visual y no cambia copy, rutas ni backend.

## Validacion agregada

- `npm outdated --json`
- `npm ls --depth=0`
- `npm audit --omit=dev`
- `npm audit`
- `npm run lint`
- `npm run typecheck`
- `npm run build`
- `npx react-doctor@latest`
- Capturas Playwright baseline/final desktop y mobile
- Comparación pixel con `sharp`
- `npm run changes:sync`

## Riesgos o arrastres

Queda pendiente ESLint 10 por incompatibilidad de plugin. Persisten advisories moderados en dependencias internas de Next y en tooling dev-only de Remotion, sin fix semver-safe disponible en esta oleada.

## Notas para presupuesto

Mantenimiento técnico mayor con migración de framework CSS, actualización de runtime React y verificación visual comparativa.
