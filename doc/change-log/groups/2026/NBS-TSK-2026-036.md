# Resumen

Grupo que recoge el refuerzo del estado `hover` del CTA `Contactar` del navbar.

## Objetivo del grupo

Hacer que el CTA principal reaccione con más contundencia sin abandonar `StarBorder` ni perder la lectura premium del navbar.

## Cambios incluidos

- fill lila de marca en `hover`
- escala `1.25` en `hover`
- actualización documental del comportamiento esperado

## Impacto en arquitectura / producto

La interacción del CTA se vuelve más protagonista sin convertir el cambio en una modificación de la primitive reusable.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- la escala pedida puede sentirse agresiva en superficies estrechas; conviene validarla visualmente

## Notas para presupuesto

Refinamiento menor de microinteracción.
