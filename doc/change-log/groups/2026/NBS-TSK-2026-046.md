# Resumen

Grupo que recoge la redefinición del hover del CTA `Contactar` hacia una inversión blanca con pausa del pulso animado.

## Objetivo del grupo

Hacer que el CTA se sienta más firme y legible al pasar por encima, sin mantener movimiento activo durante el foco.

## Cambios incluidos

- fondo blanco en hover
- texto negro en hover
- borde blanco en hover
- pulso animado pausado durante hover
- mantenimiento de la escala `1.05`

## Impacto en arquitectura / producto

El CTA gana un estado hover más limpio y la primitive `StarBorder` expone un control más reusable sobre la animación.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; el cambio solo afecta a consumers que definan la variable de pausa en hover

## Notas para presupuesto

Refinamiento menor de interacción.
