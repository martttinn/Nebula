# Resumen

Grupo que recoge el cambio de naming de las cuatro cards del carrusel de servicios de la home.

## Objetivo del grupo

Alinear el catálogo visible con los títulos pedidos por Martín: móvil, web, evolución continua y consultoría/digitalización.

## Cambios incluidos

- actualización de `data/services.ts` para reflejar los nuevos títulos y slugs visibles
- ajuste del copy corto de la primera card para que el nuevo título `Desarrollo móvil` quede semánticamente coherente
- actualización de la referencia técnica y del `decisions-log`

## Impacto en arquitectura / producto

La sección conserva su estructura y mecánica, pero cambia el framing comercial con el que presenta las capacidades del estudio.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- cualquier futura navegación o página de detalle basada en slugs deberá usar ya `desarrollo-movil`, `desarrollo-web` y `consultoria-digitalizacion`

## Notas para presupuesto

Ajuste de naming comercial en una surface pública clave del sitio.
