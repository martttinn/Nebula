# Resumen

Grupo que recoge el ajuste del naming y del orden de los links visibles del navbar.

## Objetivo del grupo

Adaptar la navegación principal al orden solicitado manteniendo intactos los destinos actuales.

## Cambios incluidos

- renombrado de `Sobre Nosotros` a `Nosotros`
- reordenación de la secuencia de links
- mantenimiento del anchor existente para la sección corporativa

## Impacto en arquitectura / producto

La navegación visible queda alineada con la nueva prioridad comercial sin tocar rutas ni estructura de secciones.

## Validación agregada

- `npm run lint`
- `npm run build`
- `npm run typecheck`

## Riesgos o arrastres

- no se esperan arrastres; el único punto a revisar más adelante sería el nombre del anchor si se renombra también la sección real

## Notas para presupuesto

Ajuste menor de navegación y copy.
