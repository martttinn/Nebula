# Resumen

Grupo que recoge la incorporación de una nueva sección pública de proyectos destacados en la home de Nebula Studios y la actualización del canon visual y documental asociada.

## Objetivo del grupo

Cerrar el salto entre “cómo trabaja el estudio” y “qué tipo de productos es capaz de construir” con una surface de portfolio premium, scroll-driven y repo-safe, sin vender todavía un portfolio live más profundo de lo que existe.

## Cambios incluidos

- creación de `components/home/projects-showcase/` como nueva familia canónica de la home
- integración de la sección después de `How we work` en `app/page.tsx`
- catálogo local de tres proyectos destacados con coreografía runtime ya activa para los tres casos y CTAs prudentes mientras no existan URLs públicas reales
- actualización de `DESIGN.md`, `.agents/decisions-log.md` y referencias técnicas/dominio para reflejar la nueva primitive
- registro de la entrega en `doc/change-log/`

## Impacto en arquitectura / producto

La home pública gana una quinta sección visible y el anchor `/#proyectos` pasa a resolver sobre una superficie real con tres proyectos encadenados. A nivel estructural, el repo incorpora una nueva carpeta canónica en `components/home/` alineada con la política actual de ownership por sección.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- los proyectos destacados usan todavía visuales conceptuales repo-safe, no capturas finales de un portfolio live
- la variante móvil/tablet simplificada sigue pendiente; el runtime actual mantiene la coreografía full-screen también fuera de desktop
- `npm run changes:sync` sigue sin poder ejecutarse porque los scripts `changes:*` no existen hoy en `package.json`, así que el índice derivado del `change-log` puede seguir desincronizado respecto al árbol real

## Notas para presupuesto

Cambio visible y claramente facturable: mezcla una nueva surface pública de alto impacto visual con trabajo de motion, responsive, arquitectura de componentes y actualización de canon documental.
