# Resumen

Grupo que recoge el acento de color aplicado a la palabra `negocio` en el claim principal del hero.

## Objetivo del grupo

Reforzar visualmente el concepto de negocio dentro del mensaje principal del hero usando el lila de marca sin romper el reveal actual.

## Cambios incluidos

- extensión ligera de `SplitText` para aceptar overrides de clase por token
- aplicación de ese override solo sobre `negocio.` en el heading del hero
- fallback equivalente en reduced motion

## Impacto en arquitectura / producto

El hero gana una palabra foco alineada con la marca y la primitive de reveal queda un poco más flexible sin alterar su contrato principal.

## Validacion agregada

- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Riesgos o arrastres

- conviene revisar visualmente el kerning y el contraste del término acentuado en los breakpoints principales

## Notas para presupuesto

Refinamiento visual localizado del hero con un pequeño endurecimiento reusable de la primitive tipográfica.
